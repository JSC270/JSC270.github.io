Mixture distribution¶

Suppose two groups of students write a test. Let $Y$ be the test score. The distribution of test scores for groups 1 and 2 are $N(70,100)$ and $N(60, 225)$. Assume that students are randomly selected from the groups. The probability a student is selected from group 1 is 0.3, and 0.7 from group 2. Simulate the distribution of $Y$.

Let $X|Z=1 \sim N(70,100), X_2|Z=0 \sim N(60,225)$, and $P(Z=1)=0.3$, where

$$ Z = \left\{ \begin{array}{ll} 1 & \mbox{if student from group 1} \\ 0 & \mbox{if student from group 2} \end{array} \right. $$

$\Diamond$ Question 7: Simulate 5000 observations $y_1,y_2,\ldots, y_{5000}$ from the distribution of $Y$.

HINT:

  1. Simulate $z_1,\ldots, z_{5000}$ from $Bin(1,0.3)$, where $Bin(n,p)$ is the binomial distribution with $n$ independent trials each with probability $p$ of success.
  2. If $z_i = 1$ then simulate $y_i$ from $ N(70,100)$, otherwise simulate $y_i$ from $ N(60,225)$

Documentation for random sampling using SciPy can be found here.

In [16]:
import numpy as np
import seaborn as sns

np.random.seed(10)
z = np.random.binomial(1, 0.3, 5000) # simulate z's

#create a list comprehension to generate a sample from y
y = [np.random.normal(70,10,1) if z1 == 1 else np.random.normal(60,15,1) for z1 in z]
y = np.concatenate(y) # concatenate arrays
print(len(y), y.mean(), y.std()) # print length, mean and sd

5000 63.198989230763424 14.358373094832453

Instead of using a list comprehension a for loop could be used.

np.random.seed(10)
z = np.random.binomial(1, 0.3, 5000) # simulate z's
y = np.zeros([5000]) # empty np array of length 5000

for x in range(len(z)):
    if z[x] == 1: 
        y[x] = np.random.normal(70, 10, 1)
    elif z[x] == 0:
        y[x] = np.random.normal(60, 15, 1)
